Custom names for tabs in Terminal: (ETA 1 min)
Sometimes, having multiple tabs gets confusing and annoying this. Naming them is quite helpful.

Add this function in ~/.bashrc.

function set-title() {
if [[ -z "$ORIG" ]]; then
ORIG=$PS1
fi
TITLE="\[\e]2;$@\a\]"
PS1=${ORIG}${TITLE}
}

Either open a new tab or run source .bashrc and type set-title <your fav-title-name>.
